## 💬 ANSWERS TO COMMON TEACHER QUESTIONS

**Q: How does the NLP work?**
A: It normalizes user input and matches keywords against a medical knowledge base, sorting by severity.

**Q: Can it be deployed online?**
A: Yes - GitHub Pages, Netlify, Vercel, or any web host. See GO_LIVE.md.

**Q: How does chat history work?**
A: localStorage API stores last 5 chats locally in browser.

**Q: Is it mobile responsive?**
A: Yes - CSS media queries for desktop, tablet, and mobile.

**Q: Why no frameworks?**
A: Project requirement was vanilla HTML/CSS/JS. Frameworks add complexity.

**Q: Can it be improved?**
A: Yes - could add backend API, real ML, audio support, etc.
